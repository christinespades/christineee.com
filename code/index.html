still rough, no sound, and the ui pixel text isn’t fully parsed or formatted yet. it’s a 3d scene with a camera, instanced meshes, a custom ui, and a skybox shader, built in windows with vulkan and c. the engine can dynamically fetch and render markdown from my repo. everything is designed to be editable and reloadable from within the engine itself.

future plans include lots of presets and shaders, post-processing and stylized effects, gizmos, selection outlines, project/map tools, level streaming, and full support for modifying terrain, meshes, lights, audio, weather, particles, ai, physics, animations, events, and gameplay systems. it’s ambitious, and i’m exploring how far i take it and how much time i dedicate to it. recent public github commits:
